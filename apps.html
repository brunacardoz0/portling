Trimeste 3:

Atividade Express
Descrição: Habilidades 1 e 2
Nesta atividade, criamos um novo projeto usando o Express e adicionamos o nodemon para facilitar o processo de desenvolvimento, configurando um script chamado "serve" para rodar o servidor na porta 5000. Em seguida, criamos duas páginas e aplicamos CSS para personalizar o visual, ajustando a cor de fundo e a cor da fonte. Foi uma excelente oportunidade para praticarmos a criação de servidores web com Express e aprimorarmos a aparência das páginas, aplicando estilos de forma simples e eficaz.

Tarefa 02
Descrição: Habilidades 3 e 4
Nessa atividade, criamos um novo projeto que aceita pacotes externos e instalamos as bibliotecas inquirer e chalk. Usamos o inquirer para solicitar o nome e a idade do usuário e, em seguida, exibimos a resposta com fundo amarelo e texto preto utilizando o chalk (com os métodos bgYellow e black). Também implementamos um tratamento de erro com catch para lidar com possíveis falhas no inquirer. Essa atividade foi uma ótima oportunidade para praticarmos o uso de pacotes externos e aprimorar o tratamento de erros nos nossos projetos.
